======================================================
         DEEP FILE UNBLOCKER - USER GUIDE
                   Version 2.5
======================================================

Thank you for installing Deep File Unblocker! This tool 
is designed to quickly fix file permissions and remove 
"Protected View" blocks from downloaded files.

--- HOW TO USE THE APP ---

1. LAUNCH THE APP
   Open "Deep File Unblocker" from your Start Menu or Desktop.
   (Note: The app requires Administrator privileges to modify 
   system permissions. Click "Yes" when prompted by Windows).

2. SELECT YOUR FILES
   - Click "Browse File" to fix a single specific file.
   - Click "Browse Folder" to fix an entire directory and 
     everything inside it.

3. CHOOSE YOUR SETTINGS
   - "Dry Run" Checkbox: Check this if you only want to TEST 
     the process. It will show you what needs fixing without 
     actually changing anything.

4. UNBLOCK & FIX
   - Click the big "Unblock & Fix Permissions" button.
   - The progress bar will show you the real-time status.
   - You can click "Pause" or "Cancel" at any time if you 
     selected a massive folder by mistake.

--- TROUBLESHOOTING ---

Q: Why are some files failing to unblock?
A: If a file is currently open in another program (like Word 
   or Excel), Windows locks it. Close the file and try again.

Q: Where are the logs?
A: You can view the real-time log at the bottom of the app. 
   Click "Export Log" to save a copy to your computer if you 
   need to review what was changed.
